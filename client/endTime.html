Set Headache End Time

Headache Tracker application assumes by default that your headache is still ongoing. That is why "Still Going On" icon is selected by default.

If your headache ended at an earlier time, please select or input that time using other icons or by tapping on the time or date.

If you make an error while entering data don't worry. You can review and update any entry before final confirmation.
